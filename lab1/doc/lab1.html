In this lab we will cover some basics of Matlab and Image Processing. The task to be performed is loading an image into Matlab and converting the image data to another colour representation (e.g. from RGB to rgb). If you are already familiar with Matlab, implement color space conversions. Visualize your results.

For those of you that have no experience with Matlab whatsoever, a general Matlab introduction can be found here. I suggest you quickly go over it to learn the basics before you go any further. For image processing, here is another brief introduction.

The command to read an image from disk is imread. The formula for the rgb colour space can be found in the syllabus. Matlab performs best while avoiding for loops. So operations should be applied on images as a whole. Use logical arrays to deal with division-by-zero (don't turn off the warnings).

Regarding the remark not to turn off warning messages, but to solve the problems, I will put some emphasis on creating nice, robust and beautiful code. As opposed to ugly hacks. So use long and descriptive variable names, don't reuse names in confusing manners and pass all arguments of a function explicitly (i.e. no global variables).

Example images can be found in the directory data, but you can also use your own images. The goal of this exercise is to experiment with the different color models; see what problems arise when you try to convert the images to these different color models and try to find the advantages and disadvantages for these models.

A Survey on Color can be found here. Several color representations are discussed here. They are also in the reader, but for those of you who don't have this reader right now, the survey can be found here.

So, for this week, you should:

  • Get familiar with Matlab, as this is the environment that will be used throughout these lab sessions.
  • Read an image from disk, and write functions to convert this image from one color space into another.
  • Implement color space conversions for RGB to normalized rgb, RGB to Opponent Color Space and RGB to HSV.

Make sure that you covered the basics correctly (show me!). If you're done, you can continue with the next session.
